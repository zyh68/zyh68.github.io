傻妞sillyGirl频繁掉线解决方法

2022-12-12

安装Screen命令

yum -y install screen

显示傻妞进程号

ps -A|grep sillyGirl

杀掉傻妞进程

kill -9 xxxxxx

创建一个名为sillyGirl的会话

screen -S sillyGirl

进入傻妞目录

cd sillyGirl

运行傻妞程序

./sillyGirl

挂机按 ctal+a+d 退出
查看screen会话

screen -ls

重新连接sillyGirl会话

screen -r sillyGirl

挂机按 ctal+a+d 退出
结束挂机会话

screen -S sillyGirl -X quit
